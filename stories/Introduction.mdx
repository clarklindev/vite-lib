# MankindUI Core

Welcome to the MankindUI Core component library documentation!

This is a comprehensive React component library built with TypeScript and Tailwind CSS, designed to provide a consistent and accessible set of UI components for modern web applications.

## Features

- **TypeScript Support**: Full TypeScript support with comprehensive type definitions
- **Tailwind CSS**: Styled with Tailwind CSS for consistent design and easy customization
- **Accessibility**: Built with accessibility in mind following ARIA guidelines
- **Dark Mode**: Built-in support for light and dark themes
- **Modular**: Import only the components you need
- **Customizable**: Highly customizable with variant-based styling

## Getting Started

To use these components in your project:

```bash
npm install @mankindui/core
```

Then import the components you need:

```javascript
import { Button, Card, Input } from '@mankindui/core';
import '@mankindui/core/styles.css';
```

## Theme Support

The library includes built-in theme support with light and dark modes. You can customize the theme by modifying the CSS custom properties or by using the theme context provider.
